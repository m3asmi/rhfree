Employee Code and Contract reference in a sequence

The employee will have a code (0001) and and a contract reference (C/2016/0001) that you can change if you wish

When you create a new employee, a code will be automatically assigned to him. The employee contract reference will also be assigned automatically following a chosen seqeunce.

To learn more about sequences and how you an change the prefix (C/%(year)s/) as you like it to be, please visit the following link in our web site How to modify sequences in odoo 9 You can contact us for any question about this module via our website. rhfree.com
