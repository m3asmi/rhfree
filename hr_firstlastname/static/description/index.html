Name = Last Name + First Name

Instead of one name field, you will have last name and first name in separate fields when creating a new employee.

In many countries Last name and First name must be in separate fields.This will allow you to make reports and legal documents which require two columns two columns. This module will allow you to have these two separate fields and also keep the original field 'name' which will be automatically filled.
